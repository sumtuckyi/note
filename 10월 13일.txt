web crawling
:여러 웹 페이지를 돌아다니며 웹 페이지 url 수집 후 indexing
web crawling process
-웹페이지 다운로드 : HTML, CSS, JS 등의 코드를 가져오는 단계
-페이지 파싱 : 다운로드 받은 코드를 분석하여 필요한 데이터를 추출하는 단계
-링크 추출 및 다른 페이지 탐색 : 다른 링크를 추출하고, 다음 페이지로 넘어가서 위의 작업을 반복
-데이터 추출 및 저장 : 분석 및 시각화를 위해 데이터를 처리하고 저장함
