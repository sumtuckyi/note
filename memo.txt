포인터 , 즉 포인터 변수는 value가 아닌 address를 저정하기 위한 변수이다. 

포인터 선언
int* p; # p는 포인터 변수임을 선언
int *p1;

# 포인터 변수 하나와, 변수 하나를 선언
int* pc, c;
# 변수에 값(value)을 할당
c=5;
# 포인터 변수에 주소(address)를 할당 -> 어떤 주소? -> c의 메모리 주소
pc = &c;
printf("%d", *pc); // 5 
※포인터 변수에 주소가 아닌 value를 할당하는 경우, 에러가 발생

*는 포인터에 저장된 value를 가져다준다.  

int* pc, c;
c=5;
pc = &c; # &c는 c에 할당된 값이 있는 메모리 주소, pc가 이 주소를 가리키게 됨 
*pc = 1;  # 해당 포인터가 가리키는 주소의 값을 1로 바꿔라
printf("%d", c) // 1
